# @dimasrakas/uikit
Sharing reusable Vue.js components with Lerna and Storybook


## Install
``` bash
# with npm
npm install

# with yarn
yarn install

```

## Development & Deployment
``` bash
# with npm
npm run storybook         #localhost
npm run build-storybook   #production

# with yarn
yarn storybook            #localhost
yarn build storybook      #production
```
