# Button

With `MDX`, we can define a story for `Button` right in the middle of our
Markdown documentation.

## Install

```bash
yarn add @dimasrakas/button

// or

npm install @dimasrakas/button
```

## Imports

```js
import { Button } from "@dimasrakas/button";
```

## Usage

<Canvas withSource="open">

      variant: "primary",
