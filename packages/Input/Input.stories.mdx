# Input

With `MDX`, we can define a story for `Button` right in the middle of our
Markdown documentation.

## Install

```bash
yarn add @dimasrakas/input

// or

npm install @dimasrakas/input
```

## Imports

```js
import { Input } from "@dimasrakas/input";
```
