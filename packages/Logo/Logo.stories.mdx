# Logo

With `MDX`, we can define a story for `Logo` right in the middle of our
Markdown documentation.

## Install

```bash
yarn add @dimasrakas/Logo

// or

npm install @dimasrakas/Logo
```

## Imports

```js
import { Logo } from "@dimasrakas/logo";
```
